◆ IS_PRESSED

#define IS_PRESSED   (!(PIND & (1<<7)))

Verifica se o botão foi pressionado (Lógica Invertida/Pull-up).

Como o botão está em pull-up, PIND & (1<<7) retorna 0 quando pressionado. O operador '!' inverte para retornar TRUE.

Retorna
1 se pressionado, 0 se solto.

Configura o ADC e os Timers para PWM.

◆ loop()

void loop ( )

Loop principal de lógica do sistema.

Realiza as seguintes tarefas:

  1. Leitura e debounce do botão de start.
  2. Verificação de dados disponíveis no rádio NRF24.
  3. Processamento do pacote de dados (Struct Controls).
  4. Atuação nos motores.
Aviso
Atualmente contém código de teste (hardcoded) sobrescrevendo a entrada de rádio.

◆ main()

int main ( void )

Ponto de entrada do programa (Setup e Loop Infinito).

Configurações de Hardware:

  • DDRB: PB0 como Saída.
  • DDRC: PC1, PC2, PC3 como Saída (LEDs).
  • DDRD: Pinos da Ponte H como Saída, Botão como Entrada.
  • Pull-up ativado no pino do botão.
  • Inicialização do Rádio NRF24L01.
Retorna
int Retorno padrão (nunca alcançado em sistemas embarcados).
